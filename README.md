# AI-Systems
1. Overview
This project explores how AI systems can support real-time decision-making in environments like drones and autonomous vehicles.

2. Modules
🔹 Decision AI
LLM-based decision engine
Input → situation
Output → recommended action
🔹 Vision AI
Image-based detection
Simulates perception layer
🔹 Communication AI
Summarizes system logs
Converts data → actionable insights

3. System Thinking:
These modules represent key layers of an intelligent system:
Perception (Vision)
Decision (LLM)
Communication (Summarization)

4. Tech Stack:
Python
OpenAI / HuggingFace
Streamlit
OpenCV
