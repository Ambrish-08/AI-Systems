# AI Intelligent Systems for Real-Time Decision Making

## Overview
This project explores how AI systems can support real-time decision-making in environments like drones and autonomous vehicles.


## Modules

### Decision AI (Completed)
- Rule-based decision engine (LLM-ready)
- Input: real-world situation
- Output: risk level + recommended action


### Vision AI (In Progress)
- Image-based perception system
- Simulates object detection



### Communication AI (In Progress)
- Converts system logs into actionable insights


## System Thinking
This project is structured as a real-world intelligent system:

- Perception Layer → Vision AI  
- Decision Layer → Decision AI  
- Communication Layer → Communication AI  

## Tech Stack
- Python  
- Streamlit  
- OpenCV 
- LLM APIs 



## Goal
To simulate how AI can support real-time decision-making in safety-critical environments.
